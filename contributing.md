# Contributing to pz-reactivex

This repo packages `lua-reactivex` as a Project Zomboid Build 42 mod (`reactivex [42]`).

## Where to contribute

- **Packaging issues (this repo):** Workshop wrapper metadata, Build 42 require paths, asset pipeline, sync/watch scripts.
- **Library implementation (upstream):** contribute to `lua-reactivex`:
  - https://github.com/christophstrasen/lua-reactivex
